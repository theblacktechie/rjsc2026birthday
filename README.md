# Happy Birthday, Raven Johnson 🏀

**An interactive data visualization tribute to Raven Johnson on her 23rd birthday March 4, 2026.**

Built by [Kris E. Smith](https://twitter.com/theblacktechie)

---

## About This Project

Raven Johnson is one of the most decorated players in South Carolina Gamecocks women's basketball history. This interactive tribute was built on her 23rd birthday, the same day the 2026 SEC Tournament opened, as a companion piece to the Senior Night data visualization series published on February 26, 2026.

The piece tells her story through a bouncing basketball. Tap the ball. Each bounce reveals a new stat from a career defined by winning at every level.

---

## What's Inside

A single self-contained `index.html` file. No frameworks. No build tools. No dependencies to install. Everything (HTML, CSS, and JavaScript) lives in one file and loads instantly.

External resources loaded via CDN:
- **Google Fonts** Barlow and Barlow Condensed

## How to Deploy on GitHub Pages

1. Create a new GitHub repository
2. Upload `index.html` to the root of the repository
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)`
5. Click **Save**
6. Your site will be live at `https://yourusername.github.io/repository-name`

Takes less than two minutes.

## Design Notes

- **Font:** Barlow and Barlow Condensed throughout — consistent with the broader Raven Johnson Career Series visual identity
- **Colors:** South Carolina garnet (`#73000A`) and gold (`#C8A84B`) as primary accents
- **Ball physics:** Designed to feel like a point guard walking the ball up the court. Slow, deliberate, high arc. Raven is a lefty, so the ball launches left on first tap.
- **Accessibility:** Ball cadence and fade transitions are intentionally slow to be safe for photosensitive users
- **Responsive:** Fully optimized for both desktop and mobile
